# Flag Simulator

3D US flag simulation on a flagpole with realistic cloth physics. Control wind direction (16-point compass) and speed (0-60 mph).

**[Live Demo](https://zebiv.com/flag-simulator/)**

## Tech

- Vanilla HTML/CSS/JavaScript
- WebGL for 3D rendering
- Verlet integration cloth simulation
- US flag image from Wikimedia Commons (public domain)
